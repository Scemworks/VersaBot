### Implementation
For Software:

# Clone Repository
```bash
git clone https://github.com/Scemworks/VersaBot.git
```
# Navigate to the project directory
```bash
cd VersaBot
```
# Installation
```bash
pip install -r requirements.txt
```
# Run
```bash
python main.py
```
# Note
* Add your bot token in the .env file before running the bot.
```plaintext
TOKEN="your_bot_token"
```
